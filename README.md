Vignette
========

Vignette is a simple CLI application for remapping Java artifacts,
using 'de-obfuscation' mappings - built using [Atlas] and [Lorenz].

## Usage

```
java -jar vignette.jar -f tsrg -m mappings.tsrg -i in.jar -o out.jar
```
